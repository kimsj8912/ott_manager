# OTT Manager (CLI)

간단한 OTT 고객/콘텐츠 관리용 콘솔 프로그램입니다.  
메모리 내 리스트 데이터를 기반으로 고객 가입, 시청 기록 조회/수정, 고객 탈퇴, 콘텐츠 판권 종료, 콘텐츠 시청 순위 집계를 수행합니다.

## 프로젝트 구성

- `main.py`
	- CLI 진입점
	- 메뉴 출력 및 사용자 입력 처리
	- 입력값 정규화/검증 후 비즈니스 함수 호출
- `customer_functions.py`
	- 고객/콘텐츠 데이터 처리 핵심 로직
	- 등록, 조회, 수정, 삭제, 판권 종료, 순위 계산/출력 함수 제공

## 실행 환경

- Python 3.12 이상
- 외부 의존성 없음

## 실행 방법

1. 프로젝트 루트에서 아래 명령 실행

```bash
python main.py
```

2. 메뉴에서 원하는 기능 선택

```text
A. 고객 신규 가입
B. 마지막 시청 기록 조회
C. 시청 기록 업데이트
D. 고객 탈퇴
E. 제공 영화 판권 종료
F. 콘텐츠 순위 출력
G. 종료
```

## 초기 데이터

- 고객 4명, 콘텐츠 4개가 코드에 하드코딩되어 시작합니다.
- 데이터 저장소(DB/파일)는 사용하지 않으며, 실행 중 메모리에서만 변경됩니다.
- 프로그램 종료 시 변경사항은 유지되지 않습니다.

## 기능 상세

### A. 고객 신규 가입

- 입력: 고객 ID, 이름, 나이, 성별, 국가, 요금제 가입 여부, 마지막 시청 제목
- 검증:
	- 나이는 숫자여야 함
	- 요금제 가입 여부는 Y/N 계열 값만 허용
	- 기존 `customer_id` 중복 시 등록 실패

### B. 마지막 시청 기록 조회

- 입력: 고객 ID
- 출력: 해당 고객의 `last_watch_title`
- 고객이 없으면 오류 메시지 출력

### C. 시청 기록 업데이트

- 입력: 고객 ID, 새 시청 제목
- 해당 고객을 찾아 `last_watch_title` 갱신

### D. 고객 탈퇴

- 입력: 고객 ID
- 목록에서 해당 고객 삭제

### E. 제공 영화 판권 종료

- 입력: 콘텐츠 제목
- 동작:
	- 콘텐츠 목록에서 제목 일치 항목 제거
	- 해당 제목을 마지막 시청으로 가진 고객들의 시청 기록을 `None`으로 초기화

### F. 콘텐츠 순위 출력

- 고객들의 마지막 시청 기록을 집계해 `(제목, 시청 수)` 형태로 정렬
- `None` 값은 집계에서 제외
- 시청 수 기준 내림차순 출력

## 핵심 함수 요약 (`customer_functions.py`)

- `register_customer(...) -> tuple[bool, str]`
	- 고객 등록 및 중복 ID 검증
- `get_last_watched(...) -> str | None`
	- 고객 마지막 시청 제목 조회
- `update_watch_history(...) -> tuple[bool, str]`
	- 시청 기록 수정
- `delete_customer(...) -> tuple[bool, str]`
	- 고객 삭제
- `end_movie_license(...) -> tuple[bool, str]`
	- 콘텐츠 제공 종료 + 고객 시청 기록 연동 정리
- `calculate_content_ranking(...) -> list[tuple[str, int]]`
	- 시청 순위 계산
- `print_content_ranking(...) -> str`
	- 순위 문자열 포맷팅

## 참고

- 현재 구현은 학습/실습용 구조입니다.
- 필요 시 다음 단계로 파일 저장(JSON/CSV) 또는 DB 연동으로 확장할 수 있습니다.
